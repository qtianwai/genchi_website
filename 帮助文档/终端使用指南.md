# 终端使用指南

## 概述

终端（Terminal）是 macOS 和 Linux 系统的命令行界面，通过输入命令来完成各种操作。本指南介绍最常用的终端命令，帮助你快速上手。

---

## 基础概念

### 什么是终端？

终端是一个文本界面，允许用户通过输入命令来与计算机交互。在 macOS 中，你可以：
- 按 `Cmd + Space` 打开 Spotlight，搜索 "Terminal" 打开终端
- 或使用 "终端" 应用

### 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + C` | 终止当前命令 |
| `Ctrl + D` | 退出当前会话（相当于 exit） |
| `Ctrl + L` | 清屏（相当于 clear） |
| `Ctrl + A` | 移动到行首 |
| `Ctrl + E` | 移动到行尾 |
| `Ctrl + U` | 清除当前行 |
| `Tab` | 自动补全命令/文件名 |
| `↑ / ↓` | 浏览历史命令 |

## 文件与目录操作

### 查看当前目录

```bash
pwd                 # 显示当前工作目录的完整路径
```

### 列出文件

```bash
ls                  # 列出当前目录的文件
ls -l              # 详细列表（显示权限、大小、日期）
ls -a              # 显示隐藏文件（以 . 开头）
ls -la             # 详细列表 + 隐藏文件
ls -lh             # 人性化大小显示（KB/MB）
ls -R              # 递归列出所有子目录
```

### 切换目录

```bash
cd 目录名           # 进入指定目录
cd ~               # 进入用户主目录
cd /               # 进入根目录
cd ..              # 返回上一级目录
cd -               # 返回上一次所在目录
```

### 创建目录

```bash
mkdir 目录名        # 创建单个目录
mkdir -p 路径/目录1/目录2  # 递归创建多级目录
```

### 删除

```bash
rm 文件名           # 删除文件（会询问确认）
rm -f 文件名       # 强制删除（不询问）
rm -r 目录名       # 递归删除目录及其内容
rm -rf 目录名      # 强制递归删除（慎用！）
rmdir 目录名       # 删除空目录
```

### 复制

```bash
cp 源文件 目标文件      # 复制文件
cp -r 源目录 目标目录   # 复制整个目录
cp -i 源 目标          # 复制时询问确认
```

### 移动/重命名

```bash
mv 源 目标         # 移动或重命名文件/目录
mv -i 源 目标      # 移动时询问确认
```

### 查看文件内容

```bash
cat 文件名         # 显示整个文件内容
head -n 行数 文件  # 显示文件前 N 行
tail -n 行数 文件  # 显示文件后 N 行
tail -f 文件       # 实时查看文件变化（常用于日志）
less 文件          # 分页查看（按 q 退出）
```

### 创建/修改文件

```bash
touch 文件名       # 创建空文件或更新修改时间
echo "内容" > 文件 # 写入内容（覆盖）
echo "内容" >> 文件 # 追加内容
nano 文件         # 文本编辑器（简单）
```

## 查找与搜索

### 查找文件

```bash
find 目录 -name "文件名"           # 按名称查找
find . -name "*.txt"              # 查找所有 txt 文件
find 目录 -type d                 # 查找目录
find 目录 -type f -size +100M     # 查找大于 100MB 的文件
```

### 搜索内容

```bash
grep "关键词" 文件            # 在文件中搜索关键词
grep -r "关键词" 目录         # 递归搜索目录中的所有文件
grep -i "关键词" 文件        # 忽略大小写搜索
grep -n "关键词" 文件        # 显示行号
grep -v "关键词" 文件        # 显示不包含关键词的行
```

## 权限管理

### 查看权限

```bash
ls -l 文件           # 查看文件权限（如 -rwxr-xr-x）
```

### 权限说明

```
drwxr-xr-x
└─ └─ └─ └─ └─ └─
  │  │  │  │  │  └─ 其他用户权限
  │  │  │  │  └──── 组用户权限
  │  │  │  └────── 所有者权限
  │  │  └───────── 目录标识（d 为目录，- 为文件）
  └─ └──────────── 类型
```

权限字符：`r`（读=4）、`w`（写=2）、`x`（执行=1）

### 修改权限

```bash
chmod 755 文件          # 常用权限：所有者全部，其他读+执行
chmod +x 文件          # 添加执行权限
chmod -x 文件          # 移除执行权限
chmod 644 文件         # 所有者读写，其他读
```

### 修改所有者

```bash
sudo chown 用户:组 文件   # 更改文件所有者和组
```

## 进程管理

### 查看进程

```bash
ps                     # 查看当前用户进程
ps aux                 # 查看所有进程（详细）
ps aux | grep 关键词    # 查找特定进程
top                    # 实时查看系统资源占用（按 q 退出）
```

### 终止进程

```bash
kill 进程ID             # 终止指定进程
kill -9 进程ID          # 强制终止（慎用）
killall 进程名          # 终止所有同名进程
```

## 网络命令

### 检查连接

```bash
ping 地址              # 测试网络连通性
ping -c 4 地址         # 只 ping 4 次
curl 地址              # 获取网页内容
wget 地址              # 下载文件
```

### SSH 远程连接

```bash
ssh 用户@服务器地址     # 连接到远程服务器
scp 文件 用户@服务器:路径  # 复制文件到远程服务器
```

## 压缩与解压

```bash
tar -cvf 名称.tar 目录   # 打包
tar -xvf 名称.tar        # 解包
tar -czvf 名称.tar.gz 目录  # 压缩
tar -xzvf 名称.tar.gz    # 解压

```

## Git 常用命令

### 基本操作

```bash
git status                    # 查看仓库状态
git add 文件                   # 暂存文件
git add .                     # 暂存所有修改
git commit -m "提交说明"       # 提交
git push                      # 推送到远程
git pull                      # 拉取远程更新
```

### 分支操作

```bash
git branch                    # 查看分支
git branch 分支名              # 创建分支
git checkout 分支名           # 切换分支
git checkout -b 分支名         # 创建并切换
git merge 分支名              # 合并分支
```

### 查看历史

```bash
git log                       # 查看提交历史
git log --oneline             # 简洁历史
git diff                      # 查看修改内容
```

## 包管理器（macOS/Linux）

### Homebrew (macOS)

```bash
brew update                   # 更新 brew
brew install 包名             # 安装软件
brew uninstall 包名           # 卸载软件
brew list                     # 列出已安装
brew search 关键词            # 搜索软件
```

### pip (Python)

```bash
pip3 install 包名             # 安装 Python 包
pip3 list                     # 列出已安装包
pip3 uninstall 包名           # 卸载包
```

### npm (Node.js)

```bash
npm install                   # 安装项目依赖
npm install -g 包名           # 全局安装
npm run 脚本                  # 运行 package.json 中的脚本
```


## 注意事项

1. **谨慎使用 rm -rf**：删除前确认路径是否正确
2. **sudo 权限**：需要管理员密码，输入时不显示是正常的
3. **Tab 补全**：多用 Tab 减少输入错误
4. **大小写敏感**：Linux/macOS 命令和文件名区分大小写
